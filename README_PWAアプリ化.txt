NITech International Student Life Map - PWAアプリ版

このフォルダをGitHub Pagesへアップロードすると、スマートフォンのホーム画面へ追加できるWebアプリ（PWA）として利用できます。

GitHub更新方法
1. このフォルダの中身をすべてGitHubリポジトリのルートへアップロードします。
2. Commit changesを押します。
3. GitHub Pagesの更新を1〜3分待ちます。
4. スマートフォンで公開URLを開きます。

インストール方法
Android / Chrome:
- 画面上部のダウンロードアイコンを押し、「インストール」を選択します。

iPhone / Safari:
- 画面上部のダウンロードアイコンを押します。
- Safariの共有ボタン → 「ホーム画面に追加」→「追加」を選択します。

注意
- OpenStreetMapの背景地図はインターネット接続が必要です。
- 店舗一覧や検索画面など、アプリ本体は一度読み込むと一部オフラインでも開けます。
- 店舗データ更新時は data/facilities.js を更新してください。
