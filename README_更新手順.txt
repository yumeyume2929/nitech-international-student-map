GitHub Pagesの更新方法
1. このフォルダの中身をすべて選択する。
2. GitHubの nitech-international-student-map リポジトリを開く。
3. Add file → Upload files を選ぶ。
4. ファイルをドラッグし、Commit changesを押す。
5. 1～3分待って公開URLをCtrl+F5で更新する。

注意
- index.htmlがリポジトリの最上位に置かれるようにしてください。
- .nojekyllを削除しないでください。
- 営業時間や対応状況は変更されるため、公式情報を確認してください。
