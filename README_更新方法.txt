スマホの地図ジャンプ修正版 v3

GitHubへ次の3ファイルをアップロードしてください。
1. index.html（リポジトリ直下）
2. js/app-mobile-v3.js（jsフォルダ）
3. css/mobile-fix-v3.css（cssフォルダ）

新しいファイル名を使っているため、古いapp.jsのキャッシュには影響されません。
更新後、GitHub Pagesの公開処理が完了するまで1～3分待ち、通常の再読み込みをしてください。

修正内容：
・スマホで施設一覧をタップしても地図を移動しない
・スマホでは詳細を画面下のシートに表示
・クラスタをタップしても地図をズーム／移動せず、その場で展開
・カテゴリ／表示範囲の切替時に自動で地図範囲を変更しない
・地図を動かすのは「表示中の全施設」ボタンと現在地ボタンだけ
