NITech International Student Life Map - 改良版

開き方:
1. index.html をダブルクリックしてください。
2. 背景地図の表示にはインターネット接続が必要です。
3. 現在地機能は、file:// で直接開くとブラウザの制限で動作しない場合があります。
   GitHub PagesなどHTTPSで公開すると利用できます。

追加した主な機能:
- 日本語・英語の切替
- 日本語名・英語名・住所の横断検索
- 大分類のカテゴリ絞り込み
- 徒歩時間順の施設一覧
- スマートフォン対応レイアウト
- 見やすいポップアップ
- Google Mapsの徒歩ルートボタン
- 公式サイトボタン
- マーカークラスタ

元のqgis2web出力HTMLは index_qgis2web_original.html として保存しています。
